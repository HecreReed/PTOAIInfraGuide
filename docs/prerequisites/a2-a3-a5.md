# A2 / A3 / A5 代际差异（深度）

## 1. 速查

| 维度 | A2 910B | A3 910C | A5 950 |
|------|---------|---------|--------|
| 生态成熟度 | 高 | 高 | 新特性快 |
| 同步 | Event | Event | Event+BufID |
| 通信 | 基线 | 基线 | 异步等增强 |
| 工具参数 | a2a3 | a2a3 | a5 |
| ptoas arch | a3 常用 | a3 | a5 |

## 2. 同步模型对比

**A3 Event：** 精确依赖边，耗 event 资源。  

**A5 BufID：** 按 buffer 程序序，减 event 压力，约束更强（嵌套等）。  

## 3. 工程实践

1. 平台字符串先定死  
2. kernel 目录常分 `a2a3/` 与 `a5/`  
3. 性能表不跨代硬比  
4. 可移植逻辑放指令层，差异关后端
